# Kpedia Scraper (韓国語辞書サイト スクレイピングツール)

韓国語辞書サイト「Kpedia (ケイペディア)」の指定したカテゴリページから見出し語情報を自動的に収集し、CSVファイルとして出力するPythonスクリプトです。

## 🌟 主な機能

* **自動ページネーション:** カテゴリ内の全ページを自動で計算し、巡回します。
* **見出し語データの抽出:** 見出し語、意味、読み方、類義語、解説、タグ（最下層カテゴリ）、単語IDを取得します。
* **csvファイルで出力:** 取得したデータを、学習に利用しやすい形でcsv形式で出力します。

## 🛠 必須環境・インストール

* **Python 3.x**
* 外部ライブラリとして `requests` と `beautifulsoup4` を使用します。

実行前に、ターミナル（コマンドプロンプト）で以下のコマンドを実行し、ライブラリをインストールしてください。

```bash
pip install requests beautifulsoup4
```

## 🚀 使い方

ターミナルを開き、スクリプト名の後ろに**抽出したいカテゴリの基本URL**を引数として渡して実行します。

**【重要】引数にするURLの注意点**
URLの末尾にあるページ番号指定（`?nCP=1` など）は**除いた状態**のURLを指定してください。（プログラム内で自動的に付与されます）

**実行例:**
```bash
python crawl_kpedia.py https://www.kpedia.jp/p/87-2
```
見出し語数が多いと時間ががかるので気長にお待ちください。

## 📁 出力ファイル

クローリングが完了すると、スクリプトと同じフォルダにCSVファイルが自動生成されます。

* **ファイル名形式:** `[カテゴリ名]_[収録単語数]words.csv` (例: `動詞_7488words.csv`)
* **出力される列:**
    1.  見出し語 (例: 가다)
    2.  意味 (例: 行く、足を運ぶ、向く、寄せる、引く)
    3.  読み方 (例: 가다、ka-da、カダ)
    4.  類義語 (複数ある場合は読点「、」区切り)
    5.  解説
    6.  タグ (最下層カテゴリの半角スペース区切り)
    7.  id (Kpedia見出し語ページURL末尾の数字)
 
## 📖 作成したcsvファイルの使いかたなど

基本的に、Ankiに読み込ませて使用することを前提としています。
例えば、「タグ」列は、カテゴリ名が半角スペース区切りで書き込まれるため、
そのままカードの「タグ」フィールドに指定すると、自動で複数のタグが付与されます。
また、「id」列の番号はKpedia上での見出し語管理番号であるため、カードテンプレートで
```
<a href="https://kpedia.jp/w/{{id}}">원본</a>
```
というふうに書くと、原本のページへのリンクを生成することができます。

## ⚠️ 注意事項・免責事項

* **アクセス負荷の軽減:** サーバーへの過度な負荷（DoS攻撃とみなされる行為）を防ぐため、各ページへのアクセス間に待機時間（`time_sleep`）を設けています。この数値を極端に小さくしたり、ゼロにしたりしないでください（5.0〜10.0秒程度を推奨します）。
* **サイト構造の変更:** Kpedia側のHTML構造やデザインがアップデートされた場合、正しくデータが抽出できなくなる可能性があります。その際はスクリプトの修正が必要です。
* **利用規約の遵守:** スクレイピングを行う際は、対象サイトの利用規約（利用制限や著作権など）を必ず確認し、常識的な範囲内でご利用ください。
