# KR-dict-scrapers

オンラインの韓国語辞書サイトから見出し語をスクレイピングし、Ankiなどの単語帳アプリに取り込み可能なCSVファイルを出力するツール群です。

## 収録ツール

1. KONEST_scraper
韓国情報サイト「コネスト(Konest)」の「今日の韓国語」コーナーから、過去に掲載された記事を一括取得します。
2. Kpedia_scraper
韓国語辞書サイト「Kpedia」の指定カテゴリから、見出し語情報を自動収集します。
3. Proton-Shielded-Automation
Docker上のUbuntu環境からProton VPN（OpenVPN）を経由し、自身のIPアドレスを秘匿しながらスクレイピングを安全に行うための実行環境です。

* 対象サーバに過剰な負荷（DoS攻撃とみなされる行為）をかけないよう、プログラム内で待機時間を設けています。数値を極端に小さくするなどの書き換えは行わないでください。
* 対象サイトのrobots.txtや利用規約（利用制限、著作権等）を確認し、取得データは私的利用の範囲内に留め、無断での公開・再配布は控えてください。
* スクレイピングは対象サイトのHTML構造に依存しているため、将来的なデザインリニューアル等のアップデートによりエラーが発生し、抽出できなくなる可能性があります。
