# Konest「今日の韓国語」スクレイパー

韓国情報サイト「コネスト(Konest)」の「今日の韓国語」コーナーから、過去に掲載された単語データを一括で取得し、CSVファイルに保存するPythonスクリプトです。

## 概要

このスクリプトは、一覧ページから順に個別の単語ページへアクセスし、以下の情報を抽出します。
取得したデータは、Excelで開いても文字化けしない形式（BOM付きUTF-8）で `todays_korean.csv` として出力されます。

* **見出し語**: 韓国語の単語やフレーズ
* **意味**: 日本語での訳
* **解説**: 単語のニュアンスや使われるシーンなどの解説文
* **例文**: 実際の会話例など（存在する場合のみ）
* **id**: KonestのURL末尾に付与されている固有ID

保存したcsvデータは、Ankiに取り込んで使用するなどしてご活用いただけます。

## 事前準備

本スクリプトを実行するには、Python 3.x がインストールされている環境と、外部ライブラリが必要です。

### 必要なライブラリのインストール

ターミナル（Mac）または コマンドプロンプト / PowerShell（Windows）を開き、以下のコマンドを実行して必要なライブラリをインストールしてください。

```bash
pip install requests beautifulsoup4
```

### 使い方
本スクリプト (scrape_konest.py) を任意のディレクトリに配置します。

ターミナル等で該当ディレクトリに移動し、以下のコマンドを実行します。

```bash
python scrape_konest.py
```
コンソール上に進行状況が表示されます。完了すると、スクリプトと同じディレクトリに todays_korean.csv が生成されます。

### ⚠️ 注意事項（必ずお読みください）
## サーバへの負荷配慮について
スクリプト内の time_sleep は、相手先サーバへ過剰な負荷をかけないための待機時間です。サイバー攻撃（DoS）と判定されるのを防ぐため、この数値を極端に小さくしたり、ゼロにしたりしないでください。

## プログラムの保守について
スクレイピングは対象サイトのHTML構造（タグやクラス名）に依存しています。将来的にサイトのデザインリニューアル等があった場合、エラーが発生して取得できなくなる可能性があります。

## 利用規約とマナー
取得したデータはご自身の学習等の「私的利用」の範囲内に留め、無断でWeb上に公開・再配布するなどの行為は控えてください。
